# ValueLens 未來開發路線圖 (Roadmap)

本文件紀錄了使用者對於影像處理管線的未來想法與待實作功能。

## 1. 影像預處理與模糊 (Preprocessing & Blur)
目前僅實作了單一的 Bilateral (雙邊濾波) 平滑化。
- [ ] **多重模糊切換**：未來考慮加入 Gaussian Blur (高斯模糊) 或 Median Blur (中值模糊)，並提供切換選項。
- [ ] **模糊參數細分**：針對不同類型的圖源，提供更細緻的模糊半徑與強度控制。
- [ ] **智慧邊緣保留**：探索更先進的邊緣保留濾波器。

## 2. 邊緣偵測進化 (Edge Detection)
目前已實作 Canny 偵測與混合比例控制。
- [ ] **顏色自定義**：目前固定為灰階/白色，未來可新增顏色選擇器，讓使用者自定義邊緣顏色（例如：亮青色、洋紅色等）。
- [ ] **演算法選擇**：除了 Canny，可以考慮加入 Sobel 或 Laplacian 算子，觀察不同算子對階調分析的影響。
- [ ] **邊緣粗細控制**：增加邊緣線條的粗細調整功能。

## 3. 灰階與彩色分析 (Color Analysis)
- [ ] **彩色 Bypass 重新設計**：目前已先移除簡單的彩色切換。未來目標是實作「帶有階調數據標註的彩色模式」，例如在彩色圖上即時顯示某點的灰階數值。
- [ ] 對於特定顏色模糊、特定顏色 Bypass 等細緻影像處理設定。
- [ ] 基於 Logic 階調邊界的邊緣檢測演算法優化。

## 4. UI/UX 優化
- [ ] **設定檔系統**：允許儲存多組濾鏡參數（例如：一組專門看細節的邊緣設定，一組專門看大色塊的模糊設定）。
- [ ] **即時預覽小窗**：在調整參數時顯示局部放大的對照預覽。
